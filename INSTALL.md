# 🛠️ วิธีติดตั้ง CookiePilot

มี 2 ทางเลือก:
- **แบบ .exe** — ไม่ต้องติดตั้ง Python เหมาะกับคนที่แค่อยากใช้บอท
- **แบบซอร์สโค้ด Python** — เหมาะกับคนที่อยากแก้/จูนโค้ด

ไม่ว่าจะเลือกแบบไหน ต้องเตรียม **LDPlayer** ให้พร้อมก่อนเหมือนกัน (ขั้นตอนที่ 1)

---

## ขั้นตอนที่ 1: ติดตั้งและตั้งค่า LDPlayer (ต้องทำก่อนเสมอ)

1. ดาวน์โหลดและติดตั้ง **LDPlayer** (แนะนำเวอร์ชัน 9 หรือ 14) จากเว็บ [ldplayer.net](https://www.ldplayer.net/)
2. เปิด LDPlayer ขึ้นมา แล้วติดตั้งเกม **Cookie Run: Kingdom / Classic** ในตัวจำลอง (ผ่าน Play Store ในตัว LDPlayer)
3. ตั้งความละเอียดจอให้ตรงกับที่บอทจูนพิกัดไว้:
   - เปิด **การตั้งค่า (⚙)** ของ LDPlayer ที่แถบด้านข้าง
   - ไปที่แท็บ **จอแสดงผล (Display)**
   - ตั้งความละเอียดเป็น **1280 × 720** และ **DPI 240**
   - กด **บันทึกและรีสตาร์ท**

   > ⚠️ พิกัดปุ่มทั้งหมดใน `config.py` จูนมาสำหรับความละเอียดนี้เท่านั้น ถ้าใช้ความละเอียดอื่นบอทจะกดพลาด

4. เปิด **ADB debugging**:
   - ไปที่แท็บ **อื่นๆ (Other settings)** ในหน้าการตั้งค่าเดียวกัน
   - เปิดสวิตช์ **ADB debugging / เชื่อมต่อ ADB** ให้เป็น "เปิด"
   - กด **บันทึกและรีสตาร์ท**
5. จำ path ของ `adb.exe` ไว้ — ปกติจะอยู่ที่ (แล้วแต่ว่าติดตั้ง LDPlayer ไว้ไดรฟ์ไหน):
   ```
   D:\LDPlayer\LDPlayer9\adb.exe
   D:\LDPlayer\LDPlayer14\adb.exe
   C:\Program Files\LDPlayer\LDPlayer9\adb.exe
   ```
   โปรแกรมมีปุ่ม **"หาอัตโนมัติ"** ที่ค้นหาไฟล์นี้ให้ได้เอง แต่ถ้าหาไม่เจอให้กรอก path เองตรงนี้
6. เข้าเกมไปให้ถึง **หน้าล็อบบี้** (มีปุ่ม Play สีเขียว) ค้างไว้ — พร้อมให้บอทเริ่มทำงาน

---

## ขั้นตอนที่ 2A: ติดตั้งแบบ .exe (แนะนำ — ไม่ต้องมี Python)

1. คัดลอกไฟล์ `dist\CookiePilot.exe` ไปไว้ที่โฟลเดอร์ไหนก็ได้ในเครื่อง (จะสร้างโฟลเดอร์ใหม่ให้ CookiePilot โดยเฉพาะก็ได้)
2. ดับเบิลคลิกเปิด `CookiePilot.exe`
   - ถ้า Windows เตือน **"Windows protected your PC"** → กด **More info** > **Run anyway**
   (เกิดเพราะไฟล์ .exe ไม่มีลายเซ็นดิจิทัล ไม่ใช่ไวรัส แต่ Windows ยังไม่รู้จักไฟล์นี้)
3. ในโปรแกรม กรอก/เลือก path ของ `adb.exe` (จากขั้นตอนที่ 1.5) แล้วกด **ทดสอบเชื่อมต่อ** — ต้องขึ้น ✅
4. เสร็จแล้ว — พร้อมกด **▶ เริ่มบอท** ได้เลย (ดูวิธีใช้เพิ่มเติมใน `README.md`)

> ไฟล์บันทึกผล (`coin_logs\`) และค่าที่จำไว้ (`settings.json`) จะถูกสร้างขึ้นในโฟลเดอร์เดียวกับตัว `.exe` โดยอัตโนมัติ ไม่ต้องสร้างเอง

### ถ้ายังไม่มี .exe (ต้อง build เอง)
ดูขั้นตอนที่ 3 (build เป็น .exe) ด้านล่าง — ต้องติดตั้ง Python ก่อนเพื่อ build

---

## ขั้นตอนที่ 2B: ติดตั้งแบบซอร์สโค้ด Python

### 1) ติดตั้ง Python
- ดาวน์โหลด Python 3.10+ จาก [python.org/downloads](https://www.python.org/downloads/)
- ตอนติดตั้ง ✅ ติ๊กเลือก **"Add python.exe to PATH"** ด้วย
- ตรวจสอบว่าติดตั้งสำเร็จ เปิด Terminal/PowerShell แล้วรัน:
  ```bash
  python --version
  ```

### 2) ดาวน์โหลดโค้ด CookiePilot
- ถ้ามี git: `git clone <repo-url>`
- หรือดาวน์โหลดเป็น .zip แล้วแตกไฟล์ไว้ที่ไหนก็ได้ เช่น `D:\my-ecom\CookiePilot`

### 3) ติดตั้ง dependencies
เปิด Terminal ที่โฟลเดอร์โปรเจกต์ (`cd` ไปที่โฟลเดอร์ `CookiePilot`) แล้วรัน:
```bash
pip install -r requirements.txt
```
ไลบรารีที่ใช้มีแค่:
- `opencv-python` — จับคู่ template / ประมวลผลภาพ
- `numpy` — คำนวณเมทริกซ์ภาพ

> GUI ใช้ `tkinter` ซึ่งมาพร้อม Python อยู่แล้ว (ถ้าติดตั้ง Python จาก python.org แบบปกติจะมีให้เลย ไม่ต้องติดตั้งเพิ่ม)

### 4) รันโปรแกรม
```bash
python app.py          # เปิด GUI (หรือดับเบิลคลิก run_gui.bat)
```
หรือรันแบบไม่มีหน้าต่าง (CLI):
```bash
python bot.py           # กด Ctrl+C เพื่อหยุด
```

---

## ขั้นตอนที่ 3 (ถ้าต้องการ): Build เป็น .exe เอง

ต้องทำขั้นตอนที่ 2B (ติดตั้ง Python + dependencies) ให้เสร็จก่อน จากนั้น:

1. ติดตั้ง PyInstaller:
   ```bash
   pip install pyinstaller
   ```
2. ดับเบิลคลิก **`build_exe.bat`** (หรือรันคำสั่งในไฟล์นั้นเองผ่าน terminal)
3. รอสักครู่ ได้ไฟล์ใหม่ที่ `dist\CookiePilot.exe`
4. คัดลอกไฟล์ `.exe` ไปวางที่ไหนก็ได้ ดับเบิลคลิกเปิดใช้ได้ทันที (ไม่ต้องมี Python ติดตั้งในเครื่องที่จะใช้)
